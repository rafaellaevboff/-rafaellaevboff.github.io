Anexar JUnit em projeto Java e fazer testes (ambiente de desenvolvimento: IntelliJ)

Em um trabalho em grupo da faculdade meu grupo escolheu utilizar essas duas tecnologias. Tivemos um pouco de dificuldade de anexar o JUnit, pois haviam muitos tutoriais de diferentes formas de o anexar. A melhor forma que achamos foi utilizando o Maven. Caso você saiba de outra forma ficaria interessada em saber também!
Para fazer os testes fiz um código simples, de uma calculadora com 5 operações. Caso deseje ver o projeto que fiz utilizando essas tecnologias basta clicar aqui.

Como fazer:

  • Acessar o site "Maven Repository", procurar por JUnit e escolher o que tem mais usuários;
  • Escolher a versão desejada do JUnit e copiar o código disponível na aba Maven;
  • Iniciar um projeto no IntelliJ com o Maven;
  • No arquivo xml que irá ser criado após o projeto ser inicializado adicionar dentro de o código copiado no site;
  • Na pasta Main->Java criar um pacote com o código que deseja testar;
  • Na pasta test->Java criar um pacote e Class;
  • Na Class da pasta test "linkar" a Class que está seu código a ser testado, dentro da class clicar com o botão direito do mouse e em Generate escolher "Test Method" e criar o código de teste;
  • Há uma documentação na internet sobre o que pode fazer com o JUnit, basta pesquisar por API + versão do JUnit que foi anexado.
