CHAVE PRIMÁRIA

Primary key
É adicionada em uma coluna da tabela, vai ficar responsável por tornar cada elemento único. Nunhum elemento pode ter uma chave primária igual.

CHAVE ESTRANGEIRA

ADD CONSTRAINT Referencia a alguma coluna de outra tabela.

ON DELETE

cascade: apaga todos os dados relacionados com o elemento selecionado restrict: não deixa apagar os dados relacionados, mas retorna uma mensagem alertando

ON UPDATE

cascade: deica alterar todos os dados relacionados com o elemento selecionado restrict: não deixa alterar os dados relacionados, mas retorna uma mensagem alertando
