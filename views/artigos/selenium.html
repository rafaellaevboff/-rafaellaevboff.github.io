SELENIUM NO CHROME

Como usar:

  • Acessar site oficial do Selenium;
  • Acessar para baixar para Chrome;
  • Baixar a extensão Selenium;
  • Acessar a extensão e começar um novo teste.

Para realizar os testes basta dar um nome para ele e indicar qual URL irá ser testada.
Rodar o teste e quando terminar abrir a aba da extensão e finalizar o teste. é possível salvar o teste como um arquivo no computador.
